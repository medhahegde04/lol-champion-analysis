# League of Legends Champion Analysis

A data analysis project exploring League of Legends champion design through base stats and class distributions.

## Data
[Patch 25.11 League of Legends Champion Data (2025) by LAH](https://www.kaggle.com/datasets/laurenainsleyhaines/patch-25-11-league-of-legends-champion-data-2025/data) from Kaggle

## Structure
- `data/` — raw CSV
- `notebooks/` — EDA notebook
